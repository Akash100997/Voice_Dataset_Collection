WUW - Voice Dataset Collection

Thank you for helping out! Just follow the directions below to submit snippets of your voice. These will be used in WUW Detection System project as a dataset for training the model.

Just follow these steps! Please try to make each recording with no background noise, if possible.

1. Press the Record button and say the word "Jasmin" within one second.(Note: Accept any pop-ups that ask you to use your computers microphone).

2. Play back or check each recording to make sure that the word is easily understood (and not truncated).

3. For each word recording that you are happy with, click the Download link next to it. (Note : Rename the file with your name.)

4. Save it in the sharing path of \\vigneshj\Sharing\To_Muthu\Dataset

5. Repeat the step with different modulations for "Jasmin" word.


Format: start recording to see sample rate
